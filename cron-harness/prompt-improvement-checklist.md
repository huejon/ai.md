# Prompt Improvement Checklist

Use when changing prompts, agent specs, skills, or command templates.

- [ ] Identify the target artifact and its current purpose.
- [ ] Preserve useful constraints and remove only redundant wording.
- [ ] Add or clarify success criteria.
- [ ] Add or clarify output format.
- [ ] Add uncertainty handling and escalation conditions.
- [ ] Add safety, injection, and external-side-effect guardrails.
- [ ] Check for conflict with `AGENTS.md` and the harness contract.
- [ ] Add or update tests/evaluation notes for material changes.
- [ ] Keep the artifact concise enough for daily use.
